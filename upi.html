What is UPI?

Unified Payments Interface (UPI) enables real-time bank-to-bank transactions using mobile applications.

UPI Transaction Flow

  1. User initiates payment
  2. Bank verifies credentials
  3. NPCI routes the request
  4. Receiver bank credits money
